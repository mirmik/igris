Detailed Description

Черная магия 4-ой ступени. Кастует что угодно во что угодно.

horrible_cast< > This is truly evil. It completely subverts C++'s type system, allowing you to cast from any class to any other class. Technically, using a union to perform the cast is undefined behaviour (even in C). But we can see if it is OK by checking that the union is the same size as each of its members. horrible_cast<> should only be used for compiler-specific workarounds. Usage is identical to reinterpret_cast<>.

This union is declared outside the horrible_cast because BCC 5.5.1 can't inline a function with a nested class, and gives a warning.
