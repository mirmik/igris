horrible\+\_\+cast$<$ $>$ This is truly evil. It completely subverts C++\textquotesingle{}s type system, allowing you to cast from any class to any other class. Technically, using a union to perform the cast is undefined behaviour (even in C). But we can see if it is OK by checking that the union is the same size as each of its members. horrible\+\_\+cast$<$$>$ should only be used for compiler-\/specific workarounds. Usage is identical to reinterpret\+\_\+cast$<$$>$.
